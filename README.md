# Bienvenido al "Programa (back-end) Foro Hub AluraLatam" 😃

## _¿En que consiste?_ ☝️
El sistema consiste en un foro donde los usuarios pueden:
- Registrar topicos
- Eliminar topicos
- Actualizar topicos
- Listar topicos
Cabe mencionar que para acceder a cualquiera de las opciones es necesario que el usuario se identifique, genere su Token y así se autentifique correctamente.
## _¿En que programa debo de ejecutarlo?_ 💻
Utilizaremos IntelliJ e Insomia para nuestros CRUD disponibles
